SIMULATION LOOP (src/main.py):
1. Parse CLI args: --ticks N (default 500), --no-viz (skip ASCII), --fast (no delay between ticks)
2. Init Grid(GRID_WIDTH, GRID_HEIGHT)
3. Spawn all species at configured counts
4. Each tick:
   a. Collect all living entities, shuffle order
   b. For each entity: call species.tick(entity, world_state) where world_state is a dict with grid, food_positions, nearby entities
   c. Process actions: MOVE (update position with wrapping), EAT (if food at position, remove food, add energy), REPRODUCE (if energy > threshold, create child), SIGNAL (species-specific)
   d. Apply energy loss (ENERGY_LOSS_PER_TICK per entity)
   e. Kill entities with energy <= 0
   f. For wolves: check ticks_since_last_meal in metadata
   g. grid.spawn_food(FOOD_SPAWN_RATE) capped at MAX_FOOD
   h. grid.decay_scents()
   i. Log population: count each species alive, count food
   j. Render ASCII frame if not --no-viz
5. After loop: generate CSV and HTML report
