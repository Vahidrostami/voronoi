# Demo: Context Engineering as a Scaling Axis

**Does computational triage — separating deterministic evidence processing from heuristic reasoning — consistently improve LLM reasoning? Does the benefit amplify in multi-agent architectures?**

A two-study design. Study 1 replicates and extends the single-agent computational triage findings (6 findings, $\eta^2_p = 0.800$). Study 2 adds multi-agent conditions (3 agents → synthesizer) to test amplification, inter-agent agreement, and cost-effectiveness. Both studies use synthetic RGM scenarios with known causal DAGs and deterministic evaluation.
