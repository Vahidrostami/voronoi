Catastrophic forgetting occurs when a neural network trained on new tasks overwrites the weights it learned for earlier tasks. The naive strategy shows the classic "forgetting cliff" — accuracy on Task 1 drops precipitously as each new task is learned. Brain-inspired strategies protect old memories while still learning new ones.
